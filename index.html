Cube Run

Cube run is a game about a cube. WOW! You control the cube and try to avoid obstacles in 15+ levels. It also includes classic mode with all the og levels. Each level includes a banger song. Obstacle types include: Box, Wall, Launchpad, and ladder. The idea for the game is from a video series by brackeys. Watch here
